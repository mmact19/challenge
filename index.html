MMAct Challenge @ CVPR2021

Dataset Download

Challenge Overview

  • MMAct Challenge 2021 will be hosted in the CVPR'21 International Challenge on Activity Recognition (ActivityNet) Workshop This challenge asks participants to propose cross-modal video action recognition/localization approaches for addressing shortcomings in visual only approaches by using MMAct Dataset. The goal of this task is to leverage both sensor-based, e.g. body-worn sensors, and vision-based modalities in ways that can overcome the limitations imposed by modality discrepancy between train (sensor + video) and test (only video) phase. Modalities used for this challenge are: Acceleration, Orientation, Gyroscope, RGB video and Keypoints. This challenge promotes an alternate point of view on how to address vision challenges through the use of cross-modal way in the hopes of expanding research on video action understanding to further leverage the sensors commonly embedded in daily-use smart devices (e.g., smartphones) as a new trial.

MMAct Features

A large-scale multi modal dataset for action understanding

7 modalities

RGB, Keypoints, Acceleration, Gyroscope, Orientation, Wi-Fi, Pressure.

1900+ Videos

Untrimmed videos with 1920x1080@30FPS

36k Instances

Average length ranges from 3-8 seconds.

37 Classes

Daily, Abnormal, Desk work actions

4 Scenes

Free space, Occlusion, Station Entrance, Desk work.

4 + 1 Views

4 survillence views + 1 egocentric view

20 Subjects

10 female, 10 male

Randomness

Collected using a semi-naturalistic collection protocol.

TASK1: Cross-Modal Trimmed Action Recognition

In this task, participants will use trimmmed clips from MMAct with paried sensor data for using. The task allows the participants to train with trimmed sensor data and trimmed video, but test on only trimmed video for action recognition. External datasets for pre-training are allowed, but it needs to be clearly documented. Vision ONLY methods are also welcome, thus sensor data is NOT the must use but recommended to be utilized.
MMAct trimmed cross-view dataset:

  • ・~18k clips from 2 views for training, ~18k clips from the other 2 views for testing.

  • ・37 action classes under 20 subjects with four scene.

  • ・MMAct untrimmed dataset and MMAct trimmed cross-scene dataset are NOT permitted in this task.

MMAct trimmed cross-scene dataset:

  • ・~24k clips from 3 scenes (Free space, Desk, Entrance) for training, ~12k clips from 2 scenes (Occlusion, Desk) for testing.

  • ・37 action classes under 20 subjects with four camera views.

  • ・MMAct untrimmed dataset and MMAct trimmed cross-view dataset are NOT permitted in this task.


TASK2: Cross-Modal Untrimmed Action Detection

In this task, participants will use untrimmed paired sensor data and video for training, test on only untrimmed videos for temporal action localization with the output being the recognized action class and its start and end time in the untrimmed video. External datasets for pre-training are allowed, but it needs to be clearly documented. Vision ONLY methods are also welcome, thus sensor data is NOT the must use but recommaned to be utilized.
MMAct untrimmed cross-session dataset:

  • ・~22k clips from for training, ~16k clips for testing.

  • ・37 action classes under 20 subjects with four camera views and four scenes.

  • ・MMAct trimmed cross-view dataset and cross-scene dataset are NOT permitted in this task.

Evaluation Metrics

Task 1. Cross-Modal Trimmed Action Recognition: The evaluation will be done across MMAct trimmed cross-view dataset and cross-scene dataset. We will use the mean Average Precision (mAP) for Top-1 as our metrics, and the winner of this challenge will be selected based on the average of these this metric across the above two datasets.
Task 2. Cross-Modal Untrimmed Action Detection: We will use the Interpolated Average Precision (AP) as our evaluation metric, which is also used by ActivityNet. The winner will be selected based on this metric evaluated on MMAct untrimmed cross-session dataset.
